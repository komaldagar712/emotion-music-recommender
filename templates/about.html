Emotion-Based Music Recommender 🎧

📌 Project Description

This project detects human emotions using facial expressions and recommends songs accordingly. It combines Computer Vision and Machine Learning to provide a personalized music experience.

⚙️ Technologies Used

  • Python
  • Flask (Backend)
  • OpenCV (Face Detection)
  • TensorFlow / Keras (CNN Model)
  • HTML, CSS, JavaScript

🧠 Working of the Project

1. Webcam captures live image
2. OpenCV detects face using Haar Cascade
3. Image is converted to grayscale
4. CNN model predicts emotion
5. Emotion is mapped to song list
6. Songs are displayed to user

🤖 Machine Learning Model

A Convolutional Neural Network (CNN) trained on FER-2013 dataset is used. It classifies emotions into: Angry, Disgust, Fear, Happy, Sad, Surprise, Neutral.

✨ Features

  • Real-time emotion detection
  • Music recommendation
  • Admin dashboard
  • History tracking

🚀 Future Scope

  • Spotify integration
  • Voice emotion detection
  • Mobile application
